# ExamenUnidad3_Patrones_BelenPerezVilla
# **Simulador de cocina con recetas modificables**
Proyecto en C# que simula una cocina con recetas modificables
---
**Patrones implementados:**
- Decorador: Nos permite que podamos agregar ingredientes extra sin la necesidad de modificar las clases originales de las recetas.
- Facade: simplifica toda la interaccion del usuario con el sistema.
